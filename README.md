# Project Overview
This project performs a complete **Exploratory Data Analysis (EDA)** 
on the COVID-19 Global Dataset covering **187 countries** to uncover 
key insights about confirmed cases, deaths, recoveries and regional 
impact worldwide.

# Dataset
- **Source:** Kaggle - Corona Virus Report
- **Rows:** 187 Countries
- **Columns:** 15
- **Features:** Confirmed, Deaths, Recovered, Active, 
WHO Region, Death Rate, Recovery Rate

# Key Business Insights

1. USA had the highest Confirmed Cases & Deaths globally
2. Americas was the most affected WHO Region
3. Average Global Death Rate was 3.02%
4. Geographically Americas, Europe & South Asia were most affected
5. Countries with High Death Rate had Low Recovery Rate (Negative Correlation)
6. Some smaller countries had surprisingly high Recovery Rates
